## [Unreleased]
### Added
- Deprecation notice in readme

## [v5.1.0] - 2020-02-21
### Added

### Changed

- Default code style fixer is now `squizlabs/php_codesniffer`

- Support for phpspec v5 and v6

### Removed
- Support for phpspec v4

### Changed

##[v4.1.0] - 2018-09-10
